# Tweveplus+ Static Website

This is a fully responsive, Google Play–style static site for Tweveplus+,
generated for GitHub Pages hosting.

**Features**
- Large app icon, rating, meta info
- About section, "What's New", scrollable screenshot carousel
- Animated Material "DOWNLOAD APP" button
- Developer contact, public footer
- Designed in pure HTML / CSS / JS
- Material Design, Google Fonts & Material Icons (CDN)
- All images and assets loaded via HTTPS/CDN
- Mobile & desktop responsive (light theme)

## Hosting

This `/public` folder is ready for GitHub Pages.  
Set repository settings to serve pages from `/public` or `/docs`.

## Preview

Open `index.html` in your browser or visit your GitHub Pages deployment.
